Wireless Hyprland

hyprctl output create headless
hyprctl keyword monitor HEADLESS-2,2560x1600@60,720x2160,1.6
wayvnc -o HEADLESS-2 0.0.0.0 5901

Wireless Sway

Temporary:

swaymsg create_output
swaymsg output "HEADLESS-1" resolution 2560x1600
wayvnc -o HEADLESS-1 0.0.0.0 5901

Permanent:

output = {
  DP-1 = {
    bg = "/nix/persist/repos/dots/wallpaper.png fill #1e1e2e";
    position = "0,0";
  };

  HEADLESS-1 = {
    mode = "2560x1600";
    position = "800,2160";
    scale = "1.6";
  };
};

Wired sway

Same thing as the wireless option, except you run:

adb reverse tcp:5901 tcp:5901

And connect to localhost on the android side
